# Проект Weather_web

## О проекте: 
API сервис, посылающий запросы к API-сервису Weather(https://github.com/Ark9119/Weather)
и отдающий пользователю обработанную информацию через WEB интерфейс.


## Лицензия: 
[![License](https://img.shields.io/badge/License-View-blue)](LICENSE)


## Возможности:
- Обрабатывает, получаемые от API данные,
производит необходимые вычисления и отдаёт пользователю в удобочитаемом, красивом виде.
- Возможность добавить типы запросов к погодному сервису(сейчас 3 типа запросов).
- Можно независмо от ответа API настраивать отображение характеристик погоды.
- Реализована регистрация для запоминания города пользователя, время сесси настраивается через куки.
- При повторных запросах используется город из базы данных API.
- Возможность изменять город.
